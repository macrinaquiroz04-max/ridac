📚 Gestionar Permisos de Tomos

Sistema avanzado para gestionar permisos individuales de tomos con control granular sobre visualización, búsqueda y exportación.

✨ Características del Sistema de Permisos de Tomos:

  • 👁️ Visualización: Control de acceso para ver el contenido de cada tomo
  • 🔍 Búsqueda: Permiso para buscar contenido específico dentro de los tomos
  • 📥 Exportación: Capacidad de descargar y exportar tomos a formato Word
  • ⚡ Asignación Rápida: Botones para configurar permisos comunes rápidamente
  • 📁 Gestión Masiva: Aplicar permisos a todos los tomos de una carpeta
  • 🎯 Control Individual: Configuración granular por cada tomo específico

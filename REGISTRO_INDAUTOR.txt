DESCRIPCIÓN DE LA OBRA
=======================
Sistema web completo desarrollado con tecnologías Python/FastAPI y JavaScript
para digitalización, análisis OCR y extracción de entidades jurídicas de 
documentos de investigación ministerial. Incluye:

- Backend API RESTful con FastAPI
- Frontend web con JavaScript vanilla
- Base de datos PostgreSQL
- Sistema de autenticación y autorización
- Módulo OCR con Tesseract y PyMuPDF
- Análisis jurídico con NLP (spaCy)
- Sistema de auditoría completo
- Gestión de tomos y carpetas de investigación
- Extracción de diligencias, personas, lugares y fechas
- Alertas de inactividad ministerial
- Visor PDF con búsqueda avanzada
